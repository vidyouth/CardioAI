# CardioAI
CardioAI is a machine learning-based heart disease prediction system that uses clinical patient data and ensemble learning models like Random Forest and XGBoost to enable early and accurate cardiovascular risk detection.

## Features
- Heart disease prediction using ML
- Data preprocessing and scaling
- Model comparison and evaluation
- ROC curve and confusion matrix visualization

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Libraries
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
